How to use

  1. Buy Blazblue Chronophantasma Extend on Steam

We support Arc System Works and everything they do. Even though Central Fiction is out on console, please support them by using a genuine copy of the game to try out blazblue-divekick

  1. Install prerequisites

This app depends on node.js being installed and availible in the system PATH. Any version should do.

  1. Download and extract the zip to a folder.

  2. Inside the folder, run the batch file setup.bat. This only needs to be done once.

  3. Start Blazblue Chronophantasma Extend

  4. Run run.bat

  5. Select any of the blazblue-divekick characters and have fun!
